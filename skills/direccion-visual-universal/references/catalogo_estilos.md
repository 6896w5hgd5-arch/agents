# Catálogo breve de estilos

Este catálogo resume rasgos observables en el carrusel de diseño gráfico revisado. No es una norma histórica ni una atribución de autor; úsalo para comparar direcciones, no para copiar una pieza concreta.

| Estilo | Rasgos útiles | Funciona bien en | Riesgos y control |
|---|---|---|---|
| **Dibujo a mano** | líneas irregulares, anotaciones, textura de papel, gesto humano | educación, conceptos, interfaces cálidas, prototipos | puede parecer improvisado; fija una escala de línea y una jerarquía |
| **Blueprint** | retícula técnica, líneas finas, fondo oscuro/azul, cotas y diagramas | documentación, ingeniería, UI técnica, ciencia ficción | baja legibilidad; reserva el detalle para capas secundarias |
| **Brutalismo** | bloques directos, tipografía grande, contraste duro, poca ornamentación | landing experimental, posters, campañas | agresivo o inaccesible; prueba contraste, foco y responsive |
| **Angel core** | luminosidad suave, halos, blancos, tonos pastel, formas etéreas | portadas, bienestar, fantasía, identidad sensible | baja separación visual o tono no apropiado; añade un color funcional y bordes claros |
| **Relieve/embossed** | volumen simulado, sombras internas, superficies táctiles | iconos, tarjetas, packaging, interfaces temáticas | sombras confusas y bajo contraste; usa relieve como acento, no para texto esencial |
| **Maximalismo** | capas, patrones, color abundante, muchos puntos focales | campañas, moodboards, escenarios, piezas editoriales | saturación; impón una cuadrícula y un foco dominante |
| **Memphis** | geometría juguetona, patrones, colores vivos, composición modular | educación, juegos, UI lúdica, material promocional | aspecto infantil o ruido; limita formas y colores funcionales |
| **Swiss/internacional** | retícula, tipografía sans, alineación, espacio negativo, jerarquía precisa | documentos, dashboards, presentaciones, interfaces | puede resultar frío; añade una señal de marca o ilustración controlada |

## Cómo combinar

- Escoge **un estilo base** para la estructura y **un acento** para personalidad. Ejemplos: Swiss + blueprint para una herramienta técnica; Swiss + dibujo a mano para una guía académica; Memphis + dibujo a mano para un juego educativo.
- No combines dos estilos que compitan por el foco (por ejemplo, maximalismo + brutalismo) sin fijar primero una jerarquía, una retícula y un límite de color.
- Conserva una capa funcional común entre medios: color semántico, tipografía legible, estados, espaciado y reglas de accesibilidad.

## Lista de control

- ¿La dirección se reconoce sin sacrificar lectura?
- ¿Hay un único foco primario por pantalla, página o pieza?
- ¿El contraste sigue siendo suficiente en escala pequeña y en impresión?
- ¿Los colores tienen significado estable y no dependen solo del tono?
- ¿El estilo puede implementarse con los recursos y el rendimiento disponibles?
- ¿La referencia se transformó en principios, en lugar de copiar elementos identificables?
